# Hospital Management System

A Windows Forms application developed with C# for managing hospital operations such as patients, doctors, appointments, and secretarial processes.

## Features
- Patient registration and login
- Doctor login and panel
- Secretary login and management panel
- Appointment creation and listing
- Branch and doctor management
- Announcement management

## User Roles
- **Patient**: Can register, log in, and create appointments
- **Doctor**: Can view appointments and patient information
- **Secretary**: Can manage doctors, branches, and appointments

## Technologies Used
- C#
- Windows Forms
- .NET Framework
- SQL Server
- Dataset & TableAdapter

## Project Structure
- Windows Forms UI
- Dataset-based database operations
- Role-based form navigation

## Notes
This project was developed for educational purposes as part of a software development course.
